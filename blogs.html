var Vs let Vs const

var is global scope .we can say function scope also.var hoisted top of the function.var value can change outside of the function also.we need to avoid it. let is function scope.we can't use let variable before initialize it.let value can re changed.if let declare inside a function we can not access it out side of the function. const is function scope.we can not re assign const value .and we can not initialize is without declare value.

Null Vs undefined

Null is a primitive type value .intentionally we assign value as Null.it avoid to creash the whole program .we explicitly assign null value.eg:let user=null undefined is also primitive value.it means we do not assign value.many ways we can get undefined value.eg:declare value without initialize.try to access object property which is not in the object.function parameters if we are not provided while calling function also we can get undefined
